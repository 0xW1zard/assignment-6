# DigiTool 🛠️
Live link: https://digitool-himel.netlify.app/

## Description
DigiTool is a simple web application for browsing and buying premium digital tools. It features a clean product catalog, transparent pricing plans, and an easy-to-use shopping cart.

## Technologies Used
* **Frontend Framework:** React (Vite)
* **Styling:** Tailwind CSS
* **UI Components:** DaisyUI
* **Notifications:** React Toastify

## 3 Key Features
1. **Dynamic Shopping Cart:** Users can seamlessly add or remove items from their cart, with the total price and item count updating automatically in real-time.
2. **Conditional Tab Navigation:** A smooth, single-page experience that allows users to instantly toggle between browsing the product catalog and reviewing their cart.
3. **Responsive Modern Design:** Built with Tailwind CSS and DaisyUI, ensuring the application looks great and functions perfectly across all device sizes.
